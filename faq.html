FAQ
01

Does the units comes with the complete system to start working immediately?

Yes, you can start from the Network-In-The-Box configuration, everything you need for small isolated network in one unit.

02

How does the system handle handover between MultiBTS systems?

BSC on a separate server. We use intra BSC handover. How to set up, described below in technical chapter.

03

How is roaming handled?

It depends on the type of core network There are several options to implement and it is better to know your current situation and requirements.

04

What is your radius coverage?

up to 200 m with 200mW (no external apmlifier), up to 1-2 km with 1W output power, up to 10 km with 10W output , NOTE: these figures are given as a reference. Coverage area will strongly depend on your local landscape and environment.

05

Do you have a specific simcards or usual simcards can be used?

Fairwaves base station could be configured to work with any SIM-cards. However it’s impossible to provide subscriber authentication and traffic encryption if you don’t know Ki secret key of your SIM-cards. If you operate a normal mobile network with encryption and subscriber authentication,you can set up only open network either get SIM-cards with known Ki keys to set up close network. You can buy a SIM-cards and program them.

06

What is capacity of the subscriber database?

A base station with one transceiver are able to handle from 200 subscribers (if you offer your subscribers a flat tariff with free local calls) to 500 (normal mode with tariffs still comfortable for subscribers). For example, with 10 000 subscribers you will need around 20 base stations. Please note this is very rough estimation.

07

Will the UMTRX work with UMTS?

UmTRX will be able to work with UMTS, but we don't have a software implementation for that.

08

How do you handle SMS and GPRS in your systems?

Fairwaves base stations are able to provide basic services: Voice and SMS also Fairwaves provide GPRS, in terms of quality it disabled in production systems to get more capacity for the voice and SMS either that’s too slow to use it as data service. We considering to use WiFi instead of GPRS.

10

Does the SGSN and GGSN are implemented?

Yes, osmo-sgsn and openggsn. This is not a complete implementation of SGSN and GGSN. For more information refer to Osmocom .

11

How to assign different external DIDs to different mobile phones?

Create multiple gateways and multiple rules in the dialplan. But we're not recommend that for this case. Usually people use SIP trunks, you need just routing appropriate calls to the trunk with a single rule.

14

How is the billing system interconnected with MNO?

It depend from the architecture but in general: fairwaves billing system can be integrated with MNO through the camel and diametr protocol.

15

Is it possible to connect external BTS to the network through A-bis interface?

We don’t support external BTS connection through A-bis interface, because of this architecture is not fault-tolerant from the connection lost. It does not have local calls switching, there is no a simple ways of backhaul optimization, that brings an extra expenses for the village connection through the VSAT.
